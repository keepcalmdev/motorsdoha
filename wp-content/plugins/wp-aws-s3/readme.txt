== Description ==

#### File manage for Amazon S3

The Amazon S3 Console is the advanvced service for Amazon S3. Unlimited files upload, Create Unlimited Bucket or Store, Create Unlimited Folder in Bucket. Manage All files in Bucket. Automatic Crop Image when upload by your Crop Dimentions and Other features.

==== KEY FEATURES ====
- All files served are hosted offsite on S3 storage. No worries about disk space!
- Create Bucket or Store.
- Select Region of Bucket when New.
- Create Multiple Folder.
- Upload any Kind of Files very easily.
- Manage Files.
- Set Multiple Crop Dimentions.
- Upload Image with crop by Dimentions.
- Get File URL and View file.

==Notes Before Use:==
- You must have an AWS account, and know your access and secret keys.
- After Config your credentials access with plugin[Go to Plugin Settings Page].
